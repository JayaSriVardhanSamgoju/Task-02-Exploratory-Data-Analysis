# Task 02: Exploratory Data Analysis (Titanic Dataset)

This repository contains a comprehensive Exploratory Data Analysis (EDA) of the Titanic dataset. The goal is to investigate passenger demographics, analyze missing values, study feature distributions, evaluate relationships between numerical variables, and identify significant factors that determined passenger survival.

---

## 🛠️ Setup and Installation

### 1. Prerequisite
Ensure you have Python 3.8+ installed on your system.

### 2. Create and Activate Virtual Environment
From this directory, run:
```bash
# Create virtual environment
python -m venv .venv

# Activate virtual environment
# On Windows (Command Prompt)
.venv\Scripts\activate.bat
# On Windows (PowerShell)
.venv\Scripts\Activate.ps1
# On macOS/Linux
source .venv/bin/activate
```

### 3. Install Dependencies
```bash
pip install -r requirements.txt
```

### 4. Execute Analysis and Generate Visualizations
To run the automated analysis script and generate all plots, tables, and reports:
```bash
python exploratory_analysis.py
```

## 💡 Key Insights Summary

1. **Gender Bias**: Females had a survival rate of **74.2%**, while males had a survival rate of only **18.9%** (supporting the historical "women and children first" protocol).
2. **Socioeconomic Bias**: Over **62.9%** of First-Class passengers survived compared to only **24.2%** of Third-Class passengers, indicating that socio-economic status was heavily linked to evacuation accessibility.
3. **Age Bias**: Children (0-12) had a survival rate of **57.9%**, confirming priority boarding for the young. Seniors (60+) had a low survival rate of **26.9%**.
4. **Outlier Fares**: Ticket fares are highly right-skewed, requiring log-transformations for machine learning tasks. A positive correlation (0.26) exists between ticket fares and survival rates.
5. **Port of Embarkation**: Passengers embarking from Cherbourg had a higher survival rate (55.4%) because a larger proportion of Cherbourg passengers booked First-Class cabins.

For a detailed analysis of findings, refer to the [observations.md](reports/observations.md) report.
